# 📝 Lista de Tareas GUI

## 📌 Descripción

Aplicación de escritorio desarrollada en Python usando Tkinter para gestionar tareas.

## 🚀 Funcionalidades

* Añadir tareas
* Marcar tareas como completadas
* Eliminar tareas
* Uso de eventos (Enter y doble clic)

## 🛠️ Tecnologías

* Python
* Tkinter

## ▶️ Cómo ejecutar

```bash
python src/todo_app.py
```
